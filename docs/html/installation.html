Installation¶

This package uses pip-uv to create an isolated environment.

Install via:

uv pip install git+git@github.com:icazzaniga/IntrinsicDimension.git

To activate the environment (inside project root):

source env/bin/activate

Dependencies¶

  • MoleculeKit (for molecular projection methods)

  • scikit-dimension (for ID estimation algorithms)

File Format Compatibility¶

MoleculeKit supports many file formats, but we recommend using:

  • .pdb for topology

  • .xtc or .dcd for trajectory

Other formats may raise parsing errors depending on MoleculeKit version. ############
